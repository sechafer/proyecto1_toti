Area de desempeño

  • Trasporte de energía (Potencia eléctrica)

    • La transmisión de energía eléctrica es el proceso de transporte de energía entre dos puntos. El transporte se lleva a cabo mediante líneas de transmisión de alta potencia, generalmente utilizando corriente alterna, que de una manera más sencilla conecta una planta con el consumidor.
